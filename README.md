

## Project Overview

This project is simple **E-commerce System** built in Java.

##  Features

- Define products with:
  - Name
  - Price
  - Quantity
- Product variations:
  - **Expirable**
  - **Non-expirabe**
  - **Shippable** 
  - **Non-shippable**

- Cart functionality:
  - Add items
  - Quantity can't exceed available stock
  - Prevent adding expired products

- Checkout functionality:
  - Calculates:
    - Subtotal
    - Shipping fees
    - Total amount
    - Customer's new balance
  - Validates:
    - Empty cart
    - Insufficient balance
    - Stock availability
    - Product expiry

- Shipping Service:
  - Collects shippable items
